# Submission

## Short Description

This project is a full-stack prediction markets application where users can create markets, place bets on different outcomes, and track results in real time.

The platform supports user and admin roles, real-time updates via polling, market resolution with proportional payout distribution, and full lifecycle management (active, resolved, archived markets).

It focuses on clean UI/UX, real-time data handling, and end-to-end functionality from database to frontend.
